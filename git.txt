git is a version control tool


Inital one time setup 
1. git init
2. git remote add origin <git url>
    eg: git remote add origin https://github.com/sagartmg2/magh-2 
   
   to check step 2  (optional)
   > git remote -v


Daily purposrse
1. git status ( optional )
2. add the files now 
    > git add <filename>
    > git add . 
3. git commit -m "js fundamentals"  
4. git push


to clone others repo
> git clone https://github.com/sagartmg2/magh-2

to creatre new branch
git checkout master
    then only
        > git checkout -b login   // login is new branch
